Extended Module player

The Extended Module Player, or xmp, is a portable command-line module player

  • Site: http://xmp.sourceforge.net
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/xmp/Makefile
  • License: GPL

Install

sudo rudix install xmp
