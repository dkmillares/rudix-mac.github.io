Axel Download Accelerator

Axel tries to accelerate HTTP/FTP downloading process by using multiple connections for one file

  • Site: http://axel.alioth.debian.org/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/axel/Makefile
  • License: GPL

Install

sudo rudix install axel
