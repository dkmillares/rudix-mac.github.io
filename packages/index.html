Security Note

You will probably get can't be opened because it is from an unidentified developer when you're trying to install some packages by web downloading. It means that the package was not registered (signed), but you can optionally trust Rudix developers and install it anyway.

Here is how to bypass once your security settings:

  1. Press the Control key when clicking the package.
  2. Choose “Open” from the pop-up menu.
  3. Click on ”Open button”.

And now follow the installation instructions.

Reference: OS X: About Gatekeeper.
