Cabextract

cabextract is Free Software for extracting Microsoft cabinet files, also called .CAB files

  • Site: http://www.cabextract.org.uk/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/cabextract/Makefile
  • License: GPL

Install

sudo rudix install cabextract
