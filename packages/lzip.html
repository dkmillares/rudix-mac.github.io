Lzip

Lzip is a lossless data compressor with a user interface similar to the one of gzip or bzip2

  • Site: http://www.nongnu.org/lzip/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/lzip/Makefile
  • License: GPL

Install

sudo rudix install lzip
