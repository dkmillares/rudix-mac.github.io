NASM

The Netwide Assembler, NASM, is an 80x86 and x86-64 assembler designed for portability and modularity

  • Site: http://www.nasm.us
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/nasm/Makefile
  • License: BSD

Install

sudo rudix install nasm
