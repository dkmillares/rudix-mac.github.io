Terminal Multiplexer

tmux is a terminal multiplexer, it enables a number of terminals (or windows) to be created, accessed, and controlled from a single screen

  • Site: http://tmux.github.io
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/tmux/Makefile
  • License: BSD

Install

sudo rudix install tmux
