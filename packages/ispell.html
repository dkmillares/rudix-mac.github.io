Ispell

International Ispell is an interactive spell-checking program which supports a large number of European languages

  • Site: https://www.cs.hmc.edu/~geoff/ispell.html
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/ispell/Makefile
  • License: BSD-Like

Install

sudo rudix install ispell
