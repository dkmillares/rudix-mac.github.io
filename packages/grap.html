Grap

Grap is a language for typesetting graphs specified and first implemented by Brian Kernighan and Jon Bentley at Bell Labs

  • Site: http://www.lunabase.org/~faber/Vault/software/grap
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/grap/Makefile
  • License: BSD

Install

sudo rudix install grap
