Python MIDI library

Python MIDI library tries to offer a reasonable object granularity to make MIDI streams a painless thing to manipulate, sequence, record, and playback

  • Site: https://github.com/vishnubob/python-midi
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/python-midi/Makefile
  • License: MIT

Install

sudo rudix install python-midi
