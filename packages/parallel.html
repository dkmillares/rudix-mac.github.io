GNU Parallel

GNU parallel is a shell tool for executing jobs concurrently locally or using remote computers

  • Site: http://savannah.gnu.org/projects/parallel/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/parallel/Makefile
  • License: GPL

Install

sudo rudix install parallel
