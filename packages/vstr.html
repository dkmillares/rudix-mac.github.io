Vstr

Vstr is a string library, it's designed so you can work optimally with readv()/writev() for input/output

  • Site: http://www.and.org/vstr/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/vstr/Makefile
  • License: LGPL

Install

sudo rudix install vstr
