XML and HTML with Python

The lxml XML toolkit is a Pythonic binding for the C libraries libxml2 and libxsl

  • Site: http://lxml.de/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/lxml/Makefile
  • License: BSD

Install

sudo rudix install lxml
