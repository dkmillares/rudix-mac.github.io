jinja2: Jinja2 (Template engine for Python)

jinja2 is a full featured template engine for Python. It has full unicode support, an optional integrated sandboxed execution environment

  • Site: http://jinja.pocoo.org
  • Port: https://github.com/rudix-mac/rudix/tree/2015/Ports/jinja2
  • License: BSD

Install

sudo rudix install jinja2
