GNU Wdiff

The GNU wdiff program is a front end to diff for comparing files on a word per word basis

  • Site: http://www.gnu.org/software/wdiff/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/wdiff/Makefile
  • License: GPL

Install

sudo rudix install wdiff
