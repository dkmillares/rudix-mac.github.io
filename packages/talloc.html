talloc: Talloc

Talloc is a hierarchical, reference counted memory pool system with destructors. It is the core memory allocator used in Samba

  • Site: http://talloc.samba.org/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/talloc/Makefile
  • License: LGPL

Install

sudo rudix install talloc
