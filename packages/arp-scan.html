arp-scan: Arp Scan

Arp-scan is a command-line tool that uses the ARP protocol to discover and fingerprint IP hosts on the local network. It is available under the GPL licence (GPLv3)

  • Site: http://www.nta-monitor.com/tools-resources/security-tools/arp-scan
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/arp-scan
  • License: GPL

Install

sudo rudix install arp-scan
