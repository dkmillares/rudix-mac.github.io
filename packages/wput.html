Wput

Wput is a command-line ftp-client that looks like wget but instead of downloading, uploads files or whole directories to remote ftp-servers

  • Site: http://wput.sourceforge.net/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/wput/Makefile
  • License: GPL

Install

sudo rudix install wput
