ABC to PostScript

abcm2ps is a command line program which converts ABC to music sheet in PostScript or SVG format

  • Site: http://moinejf.free.fr
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/abcm2ps/Makefile
  • License: GPL

Install

sudo rudix install abcm2ps
