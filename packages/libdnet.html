libdnet: Dumb Networking Library

Libdnet provides a simplified, portable interface to several low-level networking routines.

  • Site: http://code.google.com/p/libdnet/
  • Port: https://github.com/rudix-mac/rudix/tree/2015/Ports/libdnet
  • License: BSD

Install

sudo rudix install libdnet
