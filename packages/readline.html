readline: GNU Readline

The GNU Readline library provides a set of functions for use by applications that allow users to edit command lines as they are typed in

  • Site: http://tiswww.case.edu/php/chet/readline/rltop.html
  • Port: https://github.com/rudix-mac/rudix/tree/2015/Ports/readline
  • License: GPL

Install

sudo rudix install readline
