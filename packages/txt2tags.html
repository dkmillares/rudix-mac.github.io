txt2tags: txt2tags

Txt2tags is a document generator. It reads a text file with minimal markup and converts it to many formats

  • Site: http://txt2tags.org/
  • Port: https://github.com/rudix-mac/rudix/tree/2015/Ports/txt2tags

Install
