GNU SASL

GNU SASL is an implementation of the Simple Authentication and Security Layer framework and a few common SASL mechanisms

  • Site: http://www.gnu.org/software/gsasl/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/gsasl/Makefile

Install
