Paramiko

Paramiko is a module for Python that implements the SSH2 protocol for secure connections to remote machines

  • Site: http://www.lag.net/paramiko/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/paramiko/Makefile
  • License: LGPL

Install

sudo rudix install paramiko
