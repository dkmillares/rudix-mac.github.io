Liboop

Liboop is a low-level event loop management library for POSIX-based operating systems

  • Site: http://www.lysator.liu.se/liboop/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/liboop/Makefile
  • License: LGPL

Install

sudo rudix install liboop
