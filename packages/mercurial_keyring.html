mercurial_keyring: Mercurial Keyring Extension

Mercurial Keyring is a Mercurial extension used to securely save authentication passwords in password databases

  • Site: http://pypi.python.org/pypi/mercurial_keyring
  • Port: https://github.com/rudix-mac/rudix/tree/2015/Ports/mercurial_keyring
  • License: GPL

Install

sudo rudix install mercurial_keyring
