Iperf

Iperf was developed by NLANR/DAST as a modern alternative for measuring maximum TCP and UDP bandwidth performance

  • Site: http://iperf.sourceforge.net/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/iperf/Makefile
  • License: BSD

Install

sudo rudix install iperf
