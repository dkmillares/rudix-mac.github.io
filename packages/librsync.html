Librsync

librsync is a free software library that implements the rsync remote-delta algorithm

  • Site: http://librsync.sourceforge.net/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/librsync/Makefile
  • License: LGPL

Install

sudo rudix install librsync
