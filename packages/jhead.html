Jpeg Header Manipulation

jhead is used to display and manipulate data contained in the Exif header of JPEG images from digital cameras

  • Site: http://www.sentex.net/~mwandel/jhead/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/jhead/Makefile
  • License: Public Domain

Install

sudo rudix install jhead
