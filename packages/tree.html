Tree

Tree is a recursive directory listing command that produces a depth indented listing of files, which is colorized ala dircolors

  • Site: http://mama.indstate.edu/users/ice/tree/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/tree/Makefile
  • License: GPLv2

Install

sudo rudix install tree
