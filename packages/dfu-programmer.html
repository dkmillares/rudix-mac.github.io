Dfu-programmer

dfu-programmer is a Device Firmware Update based USB programmer for Atmel chips with a USB bootloader

  • Site: http://dfu-programmer.sourceforge.net/
  • Port: https://github.com/rudix-mac/rudix/tree/master/Ports/dfu-programmer/Makefile
  • License: GPL

Install

sudo rudix install dfu-programmer
