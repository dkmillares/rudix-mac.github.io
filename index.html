Rudix is a collection of pre-built Unix software delivered as packages for macOS.

Why you should use it?

  • Compatible and tested with OS X El Capitan (version 10.11);
  • Ready to use, don't need to compile anything;
  • Included all dependencies (static linked);
  • Developer friendly.

We like to say that Rudix is handcrafted work, much like the artisans do with materials we do with software packaging.
You can help to contribute by donating an amount – no donation is too small – via PayPal.

If you don't contribute, Rudix will die! Why? Well, the machine used to build packages is almost ten years old and can't be upgraded to the latest macOS version. So, help this project to continue.
