Welcome to Rudix!

Rudix is a collection of pre-built software delivered as packages. The hassle-free way to get Unix programs on OS X.

Reasons to choose:

  • Ready to use, don't need to compile anything.
  • Compatible and tested on El Capitan (OS X 10.11.6).
  • Included all libraries dependencies when applicable (linked static).
  • A wide selection of packages.
  • Comes with core Python modules.
  • Developer friendly.
